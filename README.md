# circular

# To Run this Project
- git clone https://github.com/yofiyonadio/circular.git
- cd circular\production
- npm install
- node server.js
- Server will run at http://localhost:3000
- 

# To Build this Project
- git clone https://github.com/yofiyonadio/circular.git
- cd circular\vuejs
- npm install
- cd ../nodejs
- npm install
- npm run build:prod
- cd ../production
- node server.js
- Server will run at http://localhost:3000
